Sorular

  1. Ne oluyor da butona bastığımda alttaki linkler görünmez ise görünür oluyor, görünür ise görünmez oluyor? Bunun mantığı nedir, nasıl yapılmış?
  2. Bunu responsive bir şekilde nasıl yapabilirdik? Masaüstü ekranında normal navbar gösterip mobil ekranda bu şekilde açılıp kapanan mobil navbar nasıl yapabiliriz?
  3. Masaüstünde düz olan, mobilde açılıp kapanabilen bir navbar yapalım ancak çalışma mantığı style değiştirerek değil class ekleyip çıkararak olsun

Cevaplar

  1. Navbarda ki class’ı "icon" olan a elementine basılınca myFunction() isimli fonksiyon çalışmaktadır.
    Bu fonksiyona gittiğimizde ise, id si “myLinks” olan elementleri x variable ına atarız. Bu variable’ın ,yani id si “myLinks” olan elementlerin, tıklanma gerçekleştikten önceki ki style display değeri kontrol edilir.
    Eğer style display değeri block ise, tam tersi olarak display none değeri verilir. Böylece görünür olan linkleri kapatmış oluruz.
    Eğer style display değeri none ise, tam tersi olarak display block değeri verilir. Böylece kapatılmış ve görünmüyor olan linkler açılarak gözükür hale gelir.

  2. Bunu style değiştirerek yapabiliriz. Masaüstü ve telefon için iki tane farklı container oluşturarak başlarız. Daha sonra, masaüstü class ına ait div i display block diyerek açık bırakırken telefona ait olan div i display none diyerek kapatırız. Son olarakta, resposive vererek max-width i 600px olan ekranlarda telefon css ine sahip divi display block yaparak görünür hale getirirken, masaüstüne ait olan div i display none diyerek kapatırız. Bu şekilde masaüstü ekranında normal navbar gösterip mobil ekranda açılıp kapanan mobil navbar elde etmiş oluruz.
